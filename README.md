# ClaimAI AI Service (Python)

Python-based AI microservice for intelligent claim analysis and fraud detection.

This service is part of the ClaimAI project and communicates with the [Java Spring Boot Backend](https://github.com/apurupa810-dotcom/claim-ai-backend).

## Features
- AI-powered claim summarization
- Fraud risk prediction
- Sentiment & anomaly detection
- FastAPI + Python

## Tech Stack
- Python 3.11
- FastAPI
- scikit-learn / Hugging Face
- Docker
